Class Mailboxes

A class for handling mailbox-related operations in the WildDuck API. This class provides methods for listing, creating, retrieving, updating, and deleting mailboxes.

  • List mailboxes for a specific user.

    This method retrieves a list of mailboxes for a user, optionally filtering based on special flags, hidden status, counters, and sizes. It returns detailed information about each mailbox.

    Parameters

    • userId: string

      The user's ID.

    • Optionalparams: {
          counters?: boolean;
          ip?: string;
          sess?: string;
          showHidden?: boolean;
          sizes?: boolean;
          specialUse?: boolean;
      }

      Query parameters for filtering mailboxes.

      • Optionalcounters?: boolean

        If true, include counters (total + unseen).

      • Optionalip?: string

        Optional IP address for logging.

      • Optionalsess?: string

        Optional session identifier for logging.

      • OptionalshowHidden?: boolean

        If true, include hidden folders.

      • Optionalsizes?: boolean

        If true, include mailbox sizes in bytes.

      • OptionalspecialUse?: boolean

        If true, include only folders with the specialUse flag set.

    Returns Promise<
        {
            results: {
                hidden: boolean;
                id: string;
                modifyIndex: number;
                name: string;
                path: string;
                retention: number;
                size?: number;
                specialUse: null
                | string;
                subscribed: boolean;
                total: number;
                unseen: number;
            }[];
            success: boolean;
        },
    >

    A promise that resolves to the mailbox list.

    Example

    // Fetch all mailboxes for a user
    const mailboxes = await mailboxes.listMailboxes("507f1f77bcf86cd799439011");

    Example

    // Fetch hidden mailboxes with counters and sizes included
    const mailboxes = await mailboxes.listMailboxes("507f1f77bcf86cd799439011", {
      showHidden: true,
      counters: true,
      sizes: true,
    });
